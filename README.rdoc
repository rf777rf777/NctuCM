這是我第二個Ruby on Rails作品，是一個簡單的社群平台，參考各種Rails的書籍以及資料所建構，部署於Heroku，前端設計尚未完成，登入需要先註冊帳號密碼，內容除了有Po文功能，另能關注其他使用者的貼文。

* Ruby 與 Rails版本
 Ruby : 2.2.0 Rails : 4.2.6

* 測試網址 http://gentle-journey-15893.herokuapp.com/
 1.請先註冊帳號密碼後，會自動登入。
 2.按下巡覽列的Home鍵，會進入發文介面，可發文，自己發的文章會被列於右方。
 3.User鍵會導向於這個網站註冊的使用者列表，點選自己之外的任何使用者可進行/取消關注。
 4.回到Home鍵，右方將會出現關注者的文章。
